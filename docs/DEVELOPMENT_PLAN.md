**当前版本**: v4.0（纯桌面端）

## 关键问题与风险

### 已知问题
1. **对比结果添加到队列**：功能缺失
2. **文件/文件夹选择对话框**：未实现

### 技术风险
1. **WNACG 网站结构变化**：HTML 解析可能失效
2. **AI API 成本控制**：大量对比时 API 费用可能较高
3. **并发下载稳定性**：大量任务时可能遇到资源竞争
4. **跨平台兼容性**：Windows 和 macOS 的托盘、通知差异

---

## 开发建议

1. **严格按照 Phase 顺序执行**，不要跳过
2. **Phase 2 是核心**，完成对比和下载页面后功能基本可用
3. **每个任务完成后立即验证**，不要堆积
4. **遇到问题先查文档**，不要盲目尝试
5. **保持代码整洁**，及时提交
6. **中文注释和日志**，保持一致性
