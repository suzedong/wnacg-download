# WNACG Downloader - 界面设计文档

## 概述

本文档描述 WNACG Downloader 桌面客户端的界面设计，包括：
- **整体布局**：侧边栏导航 + 自定义标题栏
- **页面设计**：搜索、对比、下载、配置页面
- **主题系统**：亮色/暗色模式
- **桌面集成**：无边框窗口、系统托盘、原生通知

**设计原则**：
- ✅ **桌面优先**：针对桌面端优化
- ✅ **紫色主题**：#667eea → #764ba2 渐变
- ✅ **暗色模式**：支持亮色/暗色切换
- ✅ **系统标题栏**：使用系统原生标题栏和控制按钮
- ✅ **轻量级**：无 UI 库依赖，所有组件自研

## 7. 桌面集成

### 7.1 无边框窗口
- 自定义标题栏
- 拖拽移动窗口
- 最小化/最大化/关闭按钮

### 7.2 系统托盘
- 托盘图标
- 托盘菜单：显示/隐藏、退出
- 最小化到托盘

### 7.3 原生通知
- 下载完成通知
- 错误通知
- 配置保存通知

### 7.4 快捷键
- `Ctrl+1`: 搜索页面
- `Ctrl+2`: 对比页面
- `Ctrl+3`: 下载页面
- `Ctrl+4`: 配置页面
- `Ctrl+D`: 切换暗色模式

### 7.5 搜索流程（自动）

搜索使用 Playwright 浏览器自动化，无 Cloudflare 问题。

**技术方案**：
- 使用 Playwright 打开真实浏览器
- 浏览器自动处理所有验证
- 等待页面加载完成后自动提取数据

**交互流程**：
1. 用户输入关键字并点击搜索
2. Playwright 自动打开浏览器进行搜索
3. 自动提取数据并显示搜索进度
4. 搜索完成，显示结果列表

## 9. 无障碍设计

### 9.1 键盘导航
- Tab: 在可聚焦元素间切换
- Enter: 触发按钮/链接
- Space: 触发复选框/按钮
- Esc: 关闭弹窗/取消操作

### 9.2 屏幕阅读器
- 所有图标按钮都有 aria-label
- 状态变化有通知
- 表单字段有关联的 label
