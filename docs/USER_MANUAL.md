## 1. 安装指南

### Windows

1. 下载 `.exe` 安装包（NSIS 格式）
2. 双击运行安装程序
3. 按照安装向导提示选择安装路径，点击"安装"
4. 安装完成后桌面会出现快捷方式，点击启动

### macOS

1. 下载 `.dmg` 磁盘镜像文件
2. 双击打开 `.dmg` 文件
3. 将 WNACG Downloader 拖入 Applications 文件夹
4. 从启动台（Launchpad）或应用程序文件夹中打开

### 首次启动提示

- 首次启动时，请允许应用访问网络（防火墙弹窗）
- 应用窗口会在屏幕中央打开，左侧为导航栏

## 2. 第一次使用

### 设置存储路径

1. 点击侧边栏 **设置** 图标（齿轮）
2. 在"存储"分区中找到 **默认存储路径**
3. 点击"选择文件夹"按钮，选择你想要保存漫画的位置
4. 路径修改后会自动保存

### 配置代理（可选）

如果你需要代理访问目标网站：

1. 在设置页面的"网络"分区中，打开 **启用代理** 开关
2. 在代理地址中输入代理 URL（例如 `http://127.0.0.1:7890`）
3. 修改后自动保存

### 设置主题

在"外观"分区中，你可以选择：

- **亮色**：浅色界面
- **暗色**：深色界面，适合夜间使用
- **跟随系统**：自动跟随操作系统的主题设置

## 3. 搜索漫画

### 搜索步骤

1. 点击侧边栏 **搜索** 图标（放大镜）
2. 在搜索框中输入关键字（漫画名称或关键词）
3. 点击 **搜索** 按钮
4. 等待搜索进度条完成（页面顶部会显示实时进度）
5. 搜索完成后，结果列表会显示所有找到的漫画

### 查看搜索结果

每个搜索结果卡片包含：

- 封面图片
- 漫画标题
- 分类（如"同人誌／漢化"）
- 图片数量
- 创建时间

### 添加到下载队列

点击任意漫画卡片右下角的 **添加到队列** 按钮，即可将该漫画加入下载队列。

### 搜索历史

1. 点击搜索框旁的 **历史** 按钮
2. 弹出模态框列出过往搜索记录
3. 每条记录显示关键字、漫画数量、搜索时间
4. 点击某条记录可加载历史搜索结果
5. 支持删除不需要的历史记录

### 重新搜索

修改搜索关键字后，再次点击搜索按钮即可获取最新数据。

## 4. 对比漫画

对比功能用于将网站搜索结果与你本地已有的漫画进行比较，避免重复下载。

### 对比步骤

1. 点击侧边栏 **对比** 图标（双箭头）
2. 选择搜索缓存文件（下拉框中选择之前搜索过的关键字）
3. 选择本地漫画文件夹（点击"选择文件夹"按钮）
4. 点击 **开始对比** 按钮

### 理解对比结果

对比完成后，你会看到两组数据：

- **需下载**：网站上有但你本地没有的漫画
- **已拥有**：你本地已经有的漫画（包含匹配方式标签）

匹配方式说明：

- **本地**：通过标题名称匹配（Levenshtein 距离算法），无需 AI 调用
- **AI**：本地无法匹配时，通过 AI 智能分析判断是否已拥有

### 添加到下载队列

在"需下载"列表中，点击 **添加到下载队列** 按钮，将这些漫画批量加入下载。

## 5. 下载漫画

### 开始下载

1. 点击侧边栏 **下载** 图标（下载箭头）
2. 下载队列中会显示待下载的漫画列表
3. 点击 **开始下载** 按钮

### 下载进度

每个下载任务显示：

- 进度条（百分比）
- 当前速度（MB/s）
- 预计剩余时间（ETA）
- 文件大小

### 控制下载

| 操作 | 说明 |
|------|------|
| **暂停** | 暂停所有下载任务，保留进度 |
| **恢复** | 继续已暂停的下载任务 |
| **取消** | 取消某个正在下载的任务 |
| **重试** | 重新尝试失败的下载任务 |

### 断点续传

如果下载中断（如网络断开），程序会自动记录已下载的部分。下次重新下载时，会从断点处继续，不会重新开始。

### 下载完成

- 下载完成后会弹出系统通知
- 文件会保存到你设置的存储路径中
- 失败的漫画会显示失败原因，可手动重试

### 并发下载

默认同时下载 3 个漫画（可在设置中调整 1-10 个）。并发数越高速度越快，但也占用更多网络带宽。

## 6. 配置管理

点击侧边栏 **设置** 图标（齿轮）进入配置页面。所有配置修改会自动保存（800ms 防抖），修改后立即生效。

### 存储

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 默认存储路径 | 漫画文件保存位置 | 程序所在目录 |

### 搜索

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 最大爬取页数 | 每次搜索的最大页数，0 表示不限制 | 0 |
| 请求间隔 | 每次请求之间的间隔时间（毫秒） | 1000ms |
| 只搜索汉化版 | 仅搜索包含"漢化"字样的漫画 | 开启 |

### 网络

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用代理 | 是否使用代理服务器 | 关闭 |
| 代理地址 | 代理服务器 URL | 空 |
| 下载源策略 | server2（最快） / worker_api（绕过 Cloudflare） | server2 |

### 下载

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 并发下载数 | 同时下载的任务数（1-10） | 3 |
| 下载重试次数 | 失败后自动重试次数 | 3 |
| 重试间隔 | 重试等待时间（秒） | 30 秒 |

### AI 匹配

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| AI API 地址 | OpenAI 兼容接口的 URL | 空 |
| AI API Key | 接口认证密钥 | 空 |
| AI 模型名称 | 模型标识符 | 空 |
| AI 温度 | 输出的创造性程度（0-2） | 0.0 |
| 匹配阈值 | 本地匹配相似度阈值（0-1） | 0.8 |

> **提示**：未配置 AI API 时，对比功能仍然正常工作——未匹配漫画会自动标记为需下载。

### 外观

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 主题 | 亮色 / 暗色 / 跟随系统 | 跟随系统 |

## 8. 故障排查

### 搜索不到漫画

1. **检查网络连接**：确认网络可以访问目标网站
2. **检查代理设置**：如果需要代理，请确认代理地址正确且已启用
3. **检查关键字**：尝试使用更通用的关键字
4. **查看日志**：搜索过程中如有错误，会在页面顶部显示

### 下载失败

1. **网络问题**：检查网络连接，尝试重新下载
2. **代理问题**：如果使用代理，确认代理是否可用
3. **磁盘空间**：确认存储路径所在磁盘有足够空间
4. **重试功能**：点击失败任务的"重试"按钮

### AI 不工作

1. **检查 API 地址**：确认 AI API URL 格式正确（如 `http://localhost:11434/v1`）
2. **检查 API Key**：如果接口需要 Key，确认已填写
3. **未配置不影响使用**：AI 未配置时，对比功能仍正常，未匹配漫画自动标记为需下载

### 软件打不开

1. **Windows**：检查是否被杀毒软件拦截，尝试右键"以管理员身份运行"
2. **macOS**：在"系统偏好设置 → 安全性与隐私"中，允许应用运行

### 下载速度慢

1. **调整并发数**：增加并发下载数（设置 → 下载 → 并发下载数）
2. **检查代理延迟**：尝试关闭代理后重试
3. **下载源策略**：切换到 `server2` 策略（最快）

### 其他问题

如果以上方法无法解决，请尝试：

1. 恢复默认配置（设置页面 → 点击"恢复默认配置"）
2. 重启应用
