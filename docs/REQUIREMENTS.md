# WNACG Downloader - 需求规格说明书

## 1. 项目概述

WNACG Downloader 是一个桌面客户端应用，支持从 wnacg.com 网站搜索、对比和下载汉化漫画。

### 1.1 产品形态

**桌面客户端**：
- **技术栈**：Tauri 2（Rust 后端）+ Vue 3（前端）
- **平台**：Windows & macOS
- **特性**：无边框窗口、系统托盘、暗色模式、侧边栏导航

### 1.2 技术栈

#### 后端（Rust）
- **异步运行时**：Tokio
- **HTTP 客户端**：reqwest
- **HTML 解析**：scraper
- **JSON 处理**：serde + serde_json
- **错误处理**：thiserror

#### 前端（Vue 3）
- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **状态管理**：Vue Composition API
- **通信**：Tauri Commands + Events

#### 桌面集成
- **框架**：Tauri 2
- **窗口**：无边框窗口
- **托盘**：系统托盘支持
- **通知**：原生系统通知

### 1.3 目标用户

- 需要批量下载汉化漫画的用户
- 希望管理本地漫画收藏的用户
- 需要智能对比本地和网站漫画的用户

## 2. 功能需求

### 2.1 搜索功能

**功能描述**：
- 通过关键字搜索 wnacg.com 网站的漫画
- 支持并发爬取多页
- 自动去重
- 保存搜索结果到本地

**用户流程**：
1. 用户在搜索页面输入关键字
2. 点击搜索按钮
3. 系统显示搜索进度
4. 搜索完成后显示结果列表
5. 用户可以选择漫画添加到下载队列

**配置项**：
- 最大爬取页数（0 表示不限制）
- 请求间隔（毫秒）
- 是否只搜索汉化版
- 代理设置

### 2.2 对比功能

**功能描述**：
- 对比搜索结果与本地漫画文件夹
- 使用 AI 智能匹配漫画名称
- 显示需要下载和已拥有的漫画

**用户流程**：
1. 用户选择搜索结果
2. 选择本地漫画文件夹
3. 点击开始对比
4. 系统扫描本地文件夹
5. AI 匹配漫画名称
6. 显示对比结果

**配置项**：
- AI 模型 API 地址
- AI API Key（可选）
- 匹配阈值（默认 0.8）

### 2.3 下载功能

**功能描述**：
- 并发下载漫画
- 支持断点续传
- 自动重试
- 下载进度实时显示

**用户流程**：
1. 用户查看下载队列
2. 点击开始下载
3. 系统显示下载进度
4. 下载完成后显示结果

**配置项**：
- 并发下载数（1-10）
- 下载重试次数
- 重试间隔（秒）
- 默认存储路径

### 2.4 配置管理

**功能描述**：
- 管理所有配置项
- 自动保存配置
- 支持恢复默认配置

**配置分类**：
- 存储设置：默认存储路径
- 网络设置：代理、爬取页数、请求间隔
- 下载设置：并发数、重试次数
- AI 设置：API 地址、密钥、阈值
- 外观设置：暗色模式、主题

---

## 3. 非功能需求

### 3.1 性能需求

- 搜索响应时间：< 30 秒（100 页以内）
- 下载速度：取决于网络带宽
- 内存占用：< 500MB
- 启动时间：< 3 秒

### 3.2 可靠性需求

- 下载失败自动重试
- 支持断点续传
- 配置自动保存
- 错误友好提示

### 3.3 用户体验需求

- 界面简洁美观
- 操作流畅
- 进度实时反馈
- 错误提示清晰

### 3.4 安全需求

- 支持 HTTPS
- API Key 加密存储
- 路径验证
- 文件权限检查

---

## 4. 界面需求

### 4.1 整体布局

- 左侧边栏导航
- 顶部自定义标题栏
- 主内容区域
- 支持暗色模式

### 4.2 页面结构

1. **搜索页面**：搜索框 + 结果列表
2. **对比页面**：搜索结果选择 + 对比结果
3. **下载页面**：下载队列 + 进度显示
4. **配置页面**：配置表单

### 4.3 交互需求

- 实时进度显示
- 系统通知
- 快捷键支持
- 拖拽支持（可选）

---

## 5. 系统需求

### 5.1 Windows

- Windows 10 或更高版本
- 64 位系统

### 5.2 macOS

- macOS 10.15 或更高版本
- Intel 或 Apple Silicon
