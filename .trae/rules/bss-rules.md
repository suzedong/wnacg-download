# WNACG Downloader - 项目规则

## 项目概述

**项目名称**：WNACG Downloader

**产品形态**：桌面客户端（Windows & macOS）

**核心功能**：搜索、对比、下载汉化漫画

**目标用户**：需要批量下载汉化漫画的用户

**技术架构**：
- **前端**：Vue 3 + TypeScript + Vite
- **后端**：Rust + Tokio（Tauri 2 框架）
- **通信**：Tauri Commands + Events
- **特性**：无边框窗口、系统托盘、暗色模式、侧边栏导航

## 功能规则

### 代理配置
- 默认使用配置文件中的代理设置
- 支持通过配置界面修改代理设置

### 搜索规则
- 爬取所有页面，无页数限制
- **不启用缓存**：每次都从网站获取最新数据
- 只搜索汉化版漫画（可通过配置包含所有漫画）
- 爬取地址：`https://www.wnacg.com/search/index.php?q={keyword}&m=&syn=yes&f=_all&s=create_time_DESC&p={page}`
- 爬取流程：
  1. 使用关键字构建搜索 URL，初始页码为 1
  2. 访问第一页，提取总页数信息
  3. 解析第一页的漫画信息（进行去重）
  4. 并行打开所有剩余页面，提高爬取效率
  5. 解析所有页面的漫画信息
  6. 保存漫画信息到文件，供对比和下载使用
- **请求间隔**：默认 **1000ms**（平衡速度和礼貌爬取）
- 数据存储：搜索结果保存到 `cache/search_{keyword}.json`
- 分类判断：通过元素的 `cate-*` 类名来判断漫画分类

### 下载规则
- 自动对比本地已有漫画，避免重复下载
- 支持并发下载，默认并发数为 3
- 下载失败时会尝试重试
- 支持断点续传
- 下载完成后校验文件完整性

### AI 匹配规则
- 使用 AI 模型进行漫画名智能匹配，提高对比准确性
- 支持远程 API 调用（OpenAI 兼容接口）
- 内置增强的相似度计算算法，处理漫画名的变体
- **启用缓存机制**，提高匹配性能
- 匹配阈值可通过配置调整，默认为 0.8
- 本地模型支持（后续实现）

## 配置管理

### 配置项清单
1. 默认存储路径
2. 代理地址（可选）
3. 最大爬取页数（默认 0，0 表示不限制）
4. **请求间隔**（毫秒，默认 1000）
5. 并发下载数（1-10，默认 3）
6. 只搜索汉化版漫画（布尔值）
7. 下载重试次数（默认 3）
8. 下载重试间隔（秒，默认 30）
9. AI 模型选择（本地/远程 API）
10. AI 模型 API 地址（如使用远程）
11. 匹配阈值（默认 0.8）

### 配置存储
- **配置文件路径**：`项目目录/.config/config.json`
- 使用 JSON 格式存储
- 用户数据和应用分离

## 核心设计理念

- **纯桌面应用**：Tauri 2 + Vue 3
- **后端**：Rust 实现核心业务逻辑（scraper, downloader, scanner, comparer, ai）
- **前端**：Vue 3 实现 UI 界面
- **通信**：Tauri Commands（调用）+ Tauri Events（通知）
- **特性**：无边框窗口、系统托盘、暗色模式、侧边栏导航
